# LP制作ウィザード

AIでLP制作する前工程を整理するための静的アプリです。

## 現在の対応範囲

- 案件ブリーフ入力
- 複数案件の一覧管理
- 新規案件作成、案件切り替え、複製、削除
- ライティング設計ページ
- デザイン設計ページ
- 入力項目右側の編集ボタン
- 入力項目単位の全体進捗表示
- ステップ別の未着手 / 入力中 / 完了表示
- 各画面下部の前後ステップ移動
- 初心者向けの複数選択式ヒアリング
- 選択肢にない内容の自由追記
- APIなしで使えるAI相談ポップアップ
- AI相談プロンプト作成、回答貼り付け、入力欄への反映
- 参考LP / 参考画像の分類
- お気に入り参考サイトの登録
- URLから取得できる範囲でサイト名・タグ候補を半自動補完
- お気に入りLP管理ページでのカテゴリー / 用途 / キーワード絞り込み
- お気に入りLP一覧専用ページ
- 参考LPの削除
- 案件内容とタグが近いお気に入り候補の表示
- ライブラリからメイン参考LP / 補助参考LPとして案件へ反映
- LP構成・原稿生成用プロンプト作成
- FV / 全セクション画像生成用プロンプト作成
- Canva Magic Layer後の編集チェックリスト
- デザインガイド生成用プロンプト作成
- ブラウザのローカルストレージ保存
- 登録・削除・入力変更時の自動保存
- JSON形式でのデータ書き出し / 読み込み
- 現在の案件のみ / 全データの書き出し

## 使い方

`LP制作ウィザードを起動.command` をダブルクリックしてください。

起動後、以下のURLが開きます。

```text
http://localhost:8787/index.html
```
